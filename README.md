# Sales Performance Dashboard | Power BI

## Project Overview
Built an interactive Power BI dashboard to analyze sales performance, profit trends, customer insights, and top-performing products.

## Features
- KPI Cards (Sales, Profit, Orders, Customers)
- Top 10 Products Analysis
- Sales Trend by Year
- Profit by Category
- Interactive Slicers
- Dynamic Filtering

## Tools Used
- Power BI
- DAX
- Power Query

## Key Insights
- Technology category generated highest profit
- Sales increased significantly in 2017
- Top products contributed major revenue share
- Profit margin maintained at 12.47%
